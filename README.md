このアプリはAIを一部使用して作られています。 

## 初期設定

本アプリは画面をスキャンして危険なキーワードを検知するため、Androidシステム側で**「他のアプリの上に重ねて表示」**と**「アクセシビリティ（ユーザー補助）」**の2つの権限を手動で許可する必要があります。

### 1. アプリ起動時の設定
1. アプリを起動し、画面内の **「設定を開く」** ボタンをタップします。
2. **「他のアプリの上に重ねて表示」** の設定画面が開くので、リストから `no-more-yamibaito` を選択し、許可を **ON** にします。

---

### Android 13以降の端末をお使いの重要な注意点
Google Playストアを介さずにAPKから直接インストールした場合、Androidのセキュリティ仕様により、アクセシビリティの設定がグレーアウトしてONにできない（「制限付き設定」という警告が出る）場合があります。
その場合は、以下の手順で制限を解除してから設定を行ってください。

#### 【制限解除の手順】
1. スマホの **「設定」アプリ** を開きます。
2. **「アプリ」** ＞ **「すべてのアプリを表示」** の順にタップします。
3. リストから **「no-more-yamibaito」** を選択します。
4. アプリ詳細画面の右上にある **「︙」(メニュー)** をタップします。
5. **「制限付き設定を許可」** （または *Allow restricted settings*）をタップし、スマホの画面ロック（パスコードや指紋など）を解除します。

---

### 2. アクセシビリティ（ユーザー補助）の有効化
上記の制限解除を行ったあと、以下の手順でサービスをONにします。

1. スマホの **「設定」** ＞ **「ユーザー補助」**（またはアクセシビリティ）を開きます。
2. **「ダウンロード済みのアプリ」**（またはインストール済みのサービス）の項目をタップします。
3. **「no-more-yamibaito」** を選択します。
4. **「no-more-yamibaito を使用する」** のスイッチを **ON** にします。
